# Simple Calculator

This is a basic calculator web application built using HTML, CSS, and JavaScript.
The project is created to practice JavaScript logic and DOM manipulation.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear all input
- Delete last digit
- Clean and simple user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How the Calculator Works
- The user enters numbers using buttons.
- An operator is selected (+, -, *, /).
- The second number is entered.
- When the equal (=) button is clicked, the calculator performs the operation.
- The result is displayed on the screen.
- The logic is written without using eval().

## Project Structure
calculator/
- index.html
- style.css
- script.js

## Purpose of This Project
This project is made for learning purposes and to understand:
- JavaScript functions
- Event handling
- Basic calculation logic
- DOM manipulation
